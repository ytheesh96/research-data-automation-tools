# Research Data Automation Tools

A public index for small chemistry and biophysics software tools.

## Overview

These tools package research workflows into clear, inspectable software: titration setup, NMR T1 relaxation fitting, DOSY diffusion calculations, radial distribution function simulation, and AI-assisted photodimerization triage. The through-line is research productivity: reduce manual calculation overhead while keeping caveats, inputs, and outputs visible.
