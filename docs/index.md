# Research Data Automation Tools

This public index is the GitHub Pages-ready version of the umbrella hub for Vaitheesh Jeypalan's research data automation tools.

## Summary for resume readers

These tools package chemistry and biophysics workflows into clear, inspectable software: titration setup, NMR T1 relaxation fitting, DOSY diffusion calculations, radial distribution function simulation, and public-safe AI-assisted photodimerization triage. The through-line is research productivity: reduce manual calculation overhead while keeping caveats, inputs, and outputs visible.
