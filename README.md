# Research Data Automation Tools


An index of small chemistry and biophysics software tools for experiment planning, data analysis, and teaching simulations. These tools package repetitive research workflows into transparent, browser-local, or dependency-light software. The through-line is practical research productivity: scientific inputs go in, calculations and visual checks stay visible, and outputs remain easy to inspect before making experimental decisions.


Current areas covered:


- titration planning and stock-solution calculations
- NMR T1 relaxation fitting
- DOSY diffusion averaging and species assignment
- radial-distribution-function teaching simulations
- photodimerization experiment planning / public-safe triage


## Tools index


| Tool | What it helps with |
| --- | --- |
| [Photodimerization Scout](https://github.com/ytheesh96/photodimerization-scout) | Plans the next irradiation experiment from substrate character, medium, organizing media, and observed result |
| [Titration Calculator](https://github.com/ytheesh96/Titration-calculator) | Plans titrations, mass/stock-solution calculations, and concentration changes |
| [T1 Calculator](https://github.com/ytheesh96/T1-Calculator) | Fits NMR T1 relaxation data from CSV integrals and visualizes fit quality |
| [DOSY Calculator](https://github.com/ytheesh96/DOSY-Calculator) | Computes weighted-average diffusion constants with species assignments and propagated errors |
| [MD Simulator](https://github.com/ytheesh96/MD_Simulator) | Visualizes radial distribution behavior for aggregated and dispersed regimes |
