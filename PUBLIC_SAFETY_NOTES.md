# Public Safety Notes

## Scope allowed in the public umbrella

- Public GitHub repository links.
- Recruiter-readable summaries of what each tool does.
- High-level descriptions of scientific workflows.
- License/status notes derived from public repository metadata.
- A publish checklist retained as an audit trail for approval and pre-publication checks.

## Scope excluded from the public umbrella

- Raw spectra, unpublished measurements, internal audit tables, private lab notes, or collaborator notes.
- Credentials, tokens, API keys, local machine paths, or shell history.
- Claims that the photodimerization scout is a validated predictive model.
- Any statement that changes ownership, institutional approval, or licensing without explicit approval.

## Component safety notes

- `hhmi-photodimerization-scout` already states a public-data boundary in its README. Keep that language prominent.
- The browser-local calculators can be described as local/client-side when their public README says so, but avoid implying medical, regulatory, or production-grade validation.
- The umbrella should remain an index of public artifacts only; do not add private research material or change licensing/ownership claims without explicit approval.
